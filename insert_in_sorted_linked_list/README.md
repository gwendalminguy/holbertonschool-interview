## Insert In Sorted Linked List

This project is about working on singly linked lists.

* [lists.h](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/insert_in_sorted_linked_list/lists.h), the header file containing all prototypes.

* [linked_lists.c](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/insert_in_sorted_linked_list/linked_lists.c), helper functions.

* [0-insert_number.c](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/insert_in_sorted_linked_list/l0-insert_number.c), a function inserting a new node in a sorted linked list.
