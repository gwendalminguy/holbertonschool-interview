## Palindrome Integer

This project is about working on palindromes.

* [palindrome.h](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/palindrome_integer/palindrome.h), the header file containing all prototypes.

* [0-is_palindrome.c](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/palindrome_integer/0-is_palindrome.c), a function checking wether an integer is a palindrome or not.
