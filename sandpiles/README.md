## Sandpiles

This project is about working on a sandpiles algorithm.

* [sandpiles.h](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/sandpiles/sandpiles.h), the header file containing all prototypes.

* [0-sandpiles.c](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/sandpiles/0-sandpiles.c), a function computing the sum of two sandpiles.
