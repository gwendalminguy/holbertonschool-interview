## Interview

This repository focuses on algorithms implementation for technical interview training.

* [Insert In Sorted Linked List](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/insert_in_sorted_linked_list), a function to insert a node in a sorted linked list, in C.

* [Lockboxes](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/lockboxes), an algorithm to determine if all lockboxes given as a list can be opened, in Python.

* [Heap Insert](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/heap_insert), a function to insert a node in a max binary heap, in C.

* [Minimum Operations](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/minimum_operations), an algorithm to determine the minimum number of operations required to transfer text, in Python.

* [Sandpiles](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/sandpiles), an algorithm to sum two sandpiles, in C.
