## Linked List Cycle

This project is about working on cycles in singly linked lists.

* [lists.h](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/linked_list_cycle/lists.h), the header file containing all prototypes.

* [0-linked_lists.c](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/linked_list_cycle/0-linked_lists.c), helper functions.

* [0-check_cycle.c](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/linked_list_cycle/0-check_cycle.c), a function checking wether a linked list contains a cycle or not.
