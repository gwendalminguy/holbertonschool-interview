## Log Parsing

This project is about working on log parsing script to monitor some server activity.

* [0-generator.py](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/log_parsing/0-generator.py), a module containing a function to randomly generate fake HTTP calls to simulate server activity.

* [0-stats.py](https://github.com/gwendalminguy/holbertonschool-interview/tree/main/log_parsing/0-stats.py), a module containing functions to analyze and log server activity.
